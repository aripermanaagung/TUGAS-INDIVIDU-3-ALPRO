Deklarasi	: i, n, fibonacci(i): integer
Deskripsi	:
input n
if i=0 atau i=1 then
cetak “fibonacci(i)=i”
else

fibonacci(i)=fibonacci(i-1)+fibonacci(i-2)
cetak fibonacci(i)

end
